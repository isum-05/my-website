#  Monkey Quest

Monkey Quest is a dynamic puzzle game where players navigate through increasingly complex mazes, overcome obstacles, and reach the goal to progress through levels.

## Features
- Dynamic maze generation using DFS algorithem
- Level-based progression system
- Reward system with coins
- Increasing difficulty per level
- Smooth and interactive gameplay
- User data management (backend integration)
- Authentication (GitHub OAuth / local user)

## Gameplay
- Navigate through the maze to reach the goal
- Avoid obstacles and dead ends
- Complete levels to earn rewards

## Tech Stack
### Frontend
- Vite (fast build tool)
- Vanilla JavaScript (OOP-based structure)
- HTML5 & CSS3
### Backend
- Node.js
- Express.js
- MongoDB (Database)
